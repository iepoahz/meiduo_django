###说明文档

###美多电商网站

###技术实现

- 基于 Python 3 + Django 框架实现

- 数据存储使用 Redis + MySQL  实现

- 第三方扩展：

    - Docker：容器

    - celery:  异步任务

    - FastDFS：自定义文件存储系统

    - Elasticsearch:  全文搜索引擎

      - 商品详情
